A custom Gutenberg block to create a notice box with different types like info, warning, success, and danger.

== Description ==

Notice Box Block is a powerful and flexible Gutenberg block that allows you to create customizable notice boxes within your WordPress posts and pages. Whether you need to display informational messages, warnings, success notifications, or error alerts, Notice Box Block has you covered.

**Features:**

* Four types of notices: Info, Warning, Success, and Danger.
* Easy to use with the Gutenberg editor.
* Customizable styles to fit your website's design.
* Lightweight and fast.

**How to Use:**

1. Install and activate the Notice Box Block plugin.
2. In the Gutenberg editor, add a new "Notice Box" block.
3. Select the type of notice you want to display (Info, Warning, Success, or Danger).
4. Add your content inside the notice box.
5. Publish or update your post/page.

== Installation ==

1. Upload the `notice-box-block` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Gutenberg editor to add and customize your notice boxes.

== Frequently Asked Questions ==

= How do I add a notice box to my post or page? =

Simply use the Gutenberg editor to add a new block, search for "Notice Box", and insert it into your content. You can then select the type of notice and add your content.

= Can I customize the appearance of the notice boxes? =

Yes, the plugin comes with default styles, but you can override them with your own custom CSS to match your website's design.

= What types of notices can I create? =

You can create four types of notices: Info, Warning, Success, and Danger.

**Compatibility:**
Notice Box Block is tested and compatible with WordPress version 5.0 and above. It requires PHP version 7.0 or higher.

**Customization:**
You can easily customize the appearance of the notice boxes by adding your own CSS styles. Use the class selectors such as `.notice-box.info`, `.notice-box.warning`, `.notice-box.success`, and `.notice-box.danger` to apply your styles.
